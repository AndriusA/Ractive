Formatters

◀ prev next ▶

Values can be formatted before rendering, using functions passed in on initialisation, which receive the value of their reference as the first argument.

As well as passing in per-instance formatter functions, there is a repository of default formatters at Ractive.formatters, to which you can add.

<p>The following text will appear in capitals, because we have defined an <code>uppercase</code> formatter:</p>
<p><strong>{{text | uppercase}}</strong></p>
var view = new Ractive({
  el: 'output',
  template: $('#template').text(),
  data: {
    text: 'capitals'
  },
  formatters: {
    uppercase: function ( input ) {
      return input.toUpperCase();
    }
  }
});
