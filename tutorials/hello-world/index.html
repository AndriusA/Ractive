A normal mustache interpolator looks like this: {{helloworld}}.

Text can be changed by calling the .set() method of a Ractive.js instance. This doesn't re-render the entire template and discard the existing elements, it just updates the text nodes that reflect changed data.

You can retrieve the current value of a reference with the get method. Below, we're using view.get('count') within a click handler.

Try running the code below (click in the console, and hit Enter), and clicking on the button.

var view = new Ractive({
  el: 'output',
  template: $('#template').text(),
  data: {
    helloworld: 'Hello world!',
    objects: 'animals',
    thing: 'motion picture',
    count: 0
  }
});

$('#button').on('click', function () {
  view.set('count', view.get( 'count' ) + 1);
});
