Conditional sections

◀ prev next ▶

A section looks like this:

{{#value}}<p>section content goes here, and can include anything</p>{{/value}}

The contents will be shown or hidden depending on the value of value – if it is truthy, it will render (empty arrays are considered falsy in this context).

Sometimes, that's the opposite of what you want – in these cases, use an inverted section:

{{^user.isSignedIn}}<a href='signin.html'>Click here to sign in</a>{{/user.isSignedIn}}
Please wait, page is loading...
// try replacing false with eg 0, 1, -1, '', 'string', [], ['stuff']... view.set('condition', false);

{{^condition}}The condition is <strong>falsy</strong>{{/condition}}
