Triples

◀ prev next ▶

Anything in a {{{triple}}}, as opposed to a regular {{mustache}}, gets inserted into the document as HTML.

(This is a powerful tool, but with great power comes great responsibility – any time you change the HTML, Ractive.js must remove the existing DOM nodes, rather than just updating text. So if you can do it with a regular {{mustache}}, you should do it that way instead.)
